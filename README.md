# Install

To install dependencies, run `npm install`.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app.

### `npm start:dev`

Runs the app in development mode, refreshing on each change.

### `npm test`

Run all tests.

### `npm test:dev`

Run all tests on change.

### `npm run build`

Builds the app for to the `build` folder.
